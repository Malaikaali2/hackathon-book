# Appendices

This section contains supplementary materials that support the main content of the book.

## Available Appendices

- **Digital Twin Workstation Specifications**: Detailed hardware requirements for simulation environments
- **Jetson Edge AI Kit Specifications**: Hardware requirements for edge AI deployment
- **Sensor Suite Documentation**: Specifications for various robotic sensors
- **Robot Lab Options**: Comparison of different robot lab configurations (budget to premium)
- **Sim-to-Real Architecture**: Technical details on bridging simulation and reality
- **Cloud-Based "Ether Lab"**: Alternative cloud-based simulation options
- **Glossary of Terms**: Definitions of key robotics and AI terminology
- **Troubleshooting Guide**: Common issues and solutions
- **Code Samples Reference**: Documentation for code examples in the book
- **Simulation Assets Guide**: Resources for creating and using simulation environments

## How to Use These Appendices

Each appendix provides detailed information that supports the concepts covered in the main modules. Use these as reference materials when implementing the concepts from the main content or when setting up your own robotic systems.
